# Contact Page - DevChallenges Solution

This is my solution to the **Contact Page** challenge from the **DevChallenges.io** platform. 
The primary goal of this project was to design and structure a responsive contact page, 
putting into practice HTML5 and CSS3 layout skills, as well as the correct handling of forms and data input elements.

## 🎯 Project Goals
- Practice responsive design and form layout using vanilla HTML and CSS.
- Learn how to work with and style form elements such as inputs, placeholders, and dropdowns.
- Gain practical experience in creating user interfaces for data capture.

## 📋 Requirements Fulfilled
According to the challenge specifications, the page includes the following fully styled elements:
- **Header & Visual Elements:** Icon, main heading, and description.
- **Structured Form:** Input fields with clear, legible placeholders.
- **Dropdown Menu:** A `select` element (dropdown) with multiple options.
- **Action Button:** An integrated and styled submit button.
- **Visual Effects:** Implementation of custom `box-shadow` values to add depth to the form card[cite: 1].

## 🛠️ Tech Stack
- **HTML5** (Semantic tags for accessibility)
- **CSS3** (Responsive design, Flexbox/Grid, and custom shadows)
- **Code Editor:** Visual Studio Code
